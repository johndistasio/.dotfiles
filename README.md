# dotfiles

Homespun artisanal application configuration, the way grandad used to do it.

## Setup

Common stuff is scripted; run:

```
$ zsh install.sh
```

## Manual setup

Site-local/personal stuff isn't scripted.

### git User Setup

Create `~/.gitconfig.local` as follows:

```
[user]
  name = "<name>"
  email = "<email>"
```
